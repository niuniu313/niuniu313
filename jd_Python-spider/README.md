# 京东商品评论爬虫

这是一个基于 [DrissionPage](https://germey.github.io/DrissionPage-docs/) 的爬虫脚本，用于抓取京东商品页面下的用户评论信息，并将其保存到 CSV 文件中。

## 🧰 使用技术

- Python 3.x
- [DrissionPage](https://github.com/germey/DrissionPage)（浏览器控制库）
- CSV 文件写入
- 简单的页面监听机制

## 📦 安装依赖

建议使用虚拟环境，安装依赖：

```bash
pip install -r requirements.txt
